❓ TROUBLESHOOTING:

  Port 80 already in use?
    → Edit .env, change HTTP_PORT=8080

  Container won't start?
    → Run: make logs (check for errors)
    → Try: make clean && make setup

  Database connection failed?
    → Run: make logs-db (check database logs)
    → Ensure database container started

  File permissions error?
    → Run: docker-compose exec app chmod -R 755 var/

    → See troubleshooting section in DOCKER_SETUP.md
